No terminal executei para instalar o phpunit versão superior a 9
docker-compose run composer require --dev phpunit/phpunit ^9

Para executar um teste
docker-compose run php vendor/bin/phpunit --colors

======================
Questão trouble-array

array("a", "b", "c", "d");
$a[] = "e";
$a[] = "f";
$a[] = "g";

Resultado: Array ( [0] => e [1] => f [2] => g )

Sem passar o índice, adiciona no fim do array o valor, por isso a resposta é A.
